## Description

- Empty state shown when no todos match current filters

## Notes

TanStack Query states:

- `isLoading` - true on first fetch when no cached data exists
- `isFetching` - true when any fetch is happening (including background refetch)
- `isPending` - mutation in progress
- `isFetched` - true after first successful fetch

We use `isLoading && !data` pattern with `useDelayedLoading` to show spinner only on initial load after 200ms delay, preventing flicker for fast loads.

Empty state only shows when `isInitialized && todosQuery.isFetched && !isLoading` to avoid flash of empty state before data loads.
